# artifacts/

| 目录 | 内容 |
|------|------|
| `checkpoints/` | 训练 run（权重、metrics、曲线） |
| `tokenizers/` | BPE 词表：`tinystories_bpe/`、`owt_bpe/` |
| `tokens/` | 编码后的 `.npy`：`tinystories_tokens/`、`owt_tokens/` |
| `plots/` | 交付图（lr / batch_size / rope 等） |
| `stories/` | 解码网格生成的故事原文 |
| `logs/` | 扫参与监控日志（`lr/`、`batch_size/`、`tokenizer/`） |
