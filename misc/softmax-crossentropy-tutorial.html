① 训练目标：交叉熵 loss（从问题出发）

问题

语言模型每步预测下一个词。怎么衡量「猜得好不好」？

我们约定用交叉熵（标准做法，不是推导出来的）：

$$L = -\sum_{r \in \{\text{猫},\text{狗},\text{追}\}} y_r \cdot \log P(r|x)$$
代入本例 · $\Sigma$ 三项全写
$$L = -\bigl[y_\text{猫}\log P(\text{猫}) + y_\text{狗}\log P(\text{狗}) + y_\text{追}\log P(\text{追})\bigr]$$
$$= -\bigl[1\cdot\log P(\text{猫}) + 0\cdot\log P(\text{狗}) + 0\cdot\log P(\text{追})\bigr]$$
$$= -\log P(\text{猫})$$

狗、追在 $\Sigma$ 里写了，但 $y=0$ 乘掉；简写 $L=-\log P(\text{真词})$ 与上式等价。

错词算不算？

标量 $L$：只剩 $-\log P(\text{猫})$。训练时：狗/追仍通过 $Z$ 和梯度 $\partial L/\partial u_\text{狗}=P(\text{狗})$ 被惩罚（见第 4 节）。

接下来缺什么？
  1. loss 里有 $P$ → 第 ② 节：softmax 从 $u$ 算出 $P$
  2. loss 里有 $\log P$ → 第 ③ 节：把 $\log P$ 改写成 $u$ 和 $Z$（方便求导）

② $P$ 从哪来？Softmax

约定 · softmax（到 $P$ 为止，没有 $\log$）
$$P(r|x) = \frac{e^{u_r}}{Z}, \qquad Z = e^{u_\text{猫}} + e^{u_\text{狗}} + e^{u_\text{追}}$$

模型输出 $u$；softmax 把 $u$ 变成概率 $P$。

1
$$Z = e^{2.0} + e^{1.0} + e^{0.0} = 7.39 + 2.72 + 1.00 = 11.11$$
2
$$P(\text{猫}) = 7.39/11.11 = 0.665,\quad P(\text{狗}) = 2.72/11.11 = 0.245,\quad P(\text{追}) = 1.00/11.11 = 0.090$$

验算：$0.665+0.245+0.090=1.000$ ✓

3

代回第 ① 节 loss：

$$L = -\log(0.665) \approx 0.408$$

③ 因为 loss 里有 $\log P$：改写成 $u$ 和 $Z$

动机（不是 softmax 定义的一部分）

第 ① 节 $L = -\log P(\text{猫})$。第 ② 节 $P = e^{u}/Z$。把两节接起来，对 $P$ 取 $\log$ 做代数变形，求导时更省事。

推导 · 从 $P=e^{u}/Z$ 出发

Step 1

$$\log P(\text{猫}) = \log\left(\frac{e^{u_\text{猫}}}{Z}\right)$$

Step 2 — $\log(a/b)=\log a - \log b$

$$= \log(e^{u_\text{猫}}) - \log Z = u_\text{猫} - \log Z$$

对狗、追同理：$\log P(\text{狗}) = 1.0 - 2.407 = -1.407$，$\log P(\text{追}) = 0.0 - 2.407 = -2.407$（$\log Z = \log 11.11 \approx 2.407$）

代回 loss · 两种写法等价
$$L = -\log P(\text{猫}) = -(u_\text{猫} - \log Z) = -u_\text{猫} + \log Z$$
$$= -2.0 + 2.407 = 0.407 \approx 0.408$$

④ 梯度 $\partial L / \partial u_i$

起点：$L = -u_\text{猫} + \log Z$，$Z = e^{u_\text{猫}} + e^{u_\text{狗}} + e^{u_\text{追}}$

对 $u_\text{猫}$

$$\frac{\partial L}{\partial u_\text{猫}} = -1 + \frac{e^{u_\text{猫}}}{Z} = P(\text{猫}) - 1 = 0.665 - 1 = -0.335$$

对 $u_\text{狗}$（$-u_\text{猫}$ 不依赖 $u_\text{狗}$，只有 $\log Z$ 有贡献）

$$\frac{\partial L}{\partial u_\text{狗}} = \frac{e^{u_\text{狗}}}{Z} = P(\text{狗}) = 0.245$$

对 $u_\text{追}$

$$\frac{\partial L}{\partial u_\text{追}} = P(\text{追}) = 0.090$$
通式
$$\frac{\partial L}{\partial u_i} = P(i|x) - y_i$$
词$P-y$
猫−0.335
狗+0.245
追+0.090

错词梯度非零 → 第 ① 节说的「错词仍被惩罚」落在这里。

附录：为啥训练不用 argmax？

推理可 argmax。训练要每个词都有梯度：softmax 给狗 $P\approx0.245$，$\partial L/\partial u_\text{狗}>0$；argmax 几乎只更新赢家。

全文链：约定 $L=-\sum y_r\log P$ → 约定 softmax 得 $P$ → 推导 $\log P=u-\log Z$ → 推导 $\partial L/\partial u_i = P-y$

CS336 · 面试常考：softmax+CE 梯度为何是 $P-y$；$e^u$ 溢出怎么办（减 max trick）

⑤ 增量：z-loss（输出稳定性）

讲义 slide · 你已会手推 softmax+CE 后，多出来的部分

增量信息 · 一句话

在交叉熵之外，再加一项罚 $\log^2 Z$，逼 softmax 分母 $Z$ 别离 1 太远 → 大模型训练更稳。PaLM 叫 z-loss。

你已经会的（复习，不是新内容）

$$\log P(r|x) = u_r - \log Z(x)$$
$$Z(x) = \sum_{r'=1}^{|V|} e^{u_{r'}(x)} \quad\text{（本教程：猫+狗+追三项）}$$

主 loss 仍是交叉熵 / 负 log 似然：让 $\log P(\text{真词})$ 尽量大。

新词

词是什么
z-loss辅助 loss，惩罚 $\log Z$ 太大（准确说：$\log^2 Z$）
$\log Z$softmax 分母取 log；$Z$ 大 → $\log Z$ 大 → 数值容易不稳
auxiliary loss辅助损失：叠在 CE 旁边，不取代 CE
$\alpha$z-loss 权重；PaLM 用 $\alpha = 10^{-4}$
stability训练不炸（loss/梯度/激活别 NaN、别飙飞）

新公式（slide + PaLM）

约定 · 总 loss = 主 loss + 辅助项

讲义写法（对每个 token $i$ 求和）：

$$L_{\text{total}} = \sum_i \Bigl[\underbrace{\log P(x_i)}_{\text{主：越大越好}} - \underbrace{\alpha \cdot \log^2 Z(x_i)}_{\text{辅：惩罚 }\log Z \text{ 偏离 }0}\Bigr]$$

PaLM 单独记辅助项名字：

$$z\_\text{loss} = 10^{-4} \cdot \log^2 Z$$

注意：slide 里主项写 $\log P$（最大化方向）；你教程里 CE 写 $L=-\log P$（最小化）。差一个负号，辅助项两边一致：都是「别让 $\log Z$ 乱飙」。

为啥要罚 $\log Z$？（直觉）

  • $Z = \sum e^{u_r}$：某个 token 的 logits 整体太大 → $Z$ 爆炸 → $e^u$ 溢出、梯度怪异
  • 罚 $\log^2 Z$ → 鼓励 $\log Z \approx 0$ → $Z \approx 1$
  • 和 CE 分工不同：CE 管「猜对词」；z-loss 管「logits 规模别失控」
  • 和 weight decay 无关：wd 罚权重；z-loss 罚 输出层 $\log Z$

数字例子（接本教程）

代入

已算：$\log Z = \log(11.11) \approx 2.407$

$$\log^2 Z \approx 2.407^2 \approx 5.79$$
$$z\_\text{loss} = 10^{-4} \times 5.79 \approx 0.00058$$

相对 $L_{\text{CE}} \approx 0.408$，很小——轻轻拽 $\log Z$ 往 0 靠。

若 logits 整体飙大，$Z=10^5$，$\log Z \approx 11.5$，$\log^2 Z \approx 132$，$z\_\text{loss} \approx 0.013$ → 罚得明显更重。

和 CE 手推的关系

手撕 CE：$P=e^u/Z$，$L=-\log P$，$\partial L/\partial u = P-y$ —— 不变。

z-loss 增量：多一项 $\alpha \log^2 Z$，反向传播时对每个 $u_r$ 多贡献一项与 $P(r)$ 有关的梯度（通过 $Z$）。面试浅层知道：辅助项抑制 logits 整体尺度 即可。

谁在用：PaLM、Baichuan 2、DCLM、OLMo 2/3 等。
记一句：CE 管对错；z-loss 管 $\log Z$ 别离 0 太远，防训练炸。
