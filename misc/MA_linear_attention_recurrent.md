# 线性注意力的递推形式（Recurrent form of linear attention）

本文自洽。读完即可看懂课上这几行式子，以及它们各自在算什么。

课上式子：

```text
标准写法里：  输出 ∝ Q Kᵀ V
线性注意力重排：Q Kᵀ V = Q (Kᵀ V)

递推：
  S_t = S_{t-1} + k_t v_tᵀ
  y_t = q_tᵀ S_t
```

下面按顺序建立这些符号，再用同一组小数把两条路算一遍，核对结果相同。

---

## 1. 场景与符号（全程共用）

一次前向里，我们有长度为 \(T\) 的序列。每个位置 \(t = 1,\ldots,T\) 上有三个向量：

| 符号 | 名字 | 形状（本笔记例子） | 含义 |
|------|------|-------------------|------|
| \(q_t\) | query | 列向量 \(d\times 1\) | 当前位置用来「去取信息」的向量 |
| \(k_t\) | key | 列向量 \(d\times 1\) | 当前位置用来「被匹配」的向量 |
| \(v_t\) | value | 列向量 \(d\times 1\) | 当前位置要贡献出去的内容 |

把整段序列排成矩阵（每行一个位置）：

| 符号 | 形状 | 怎么由 \(q_t,k_t,v_t\) 拼成 |
|------|------|---------------------------|
| \(Q\) | \(T\times d\) | 第 \(t\) 行是 \(q_tᵀ\) |
| \(K\) | \(T\times d\) | 第 \(t\) 行是 \(k_tᵀ\) |
| \(V\) | \(T\times d\) | 第 \(t\) 行是 \(v_tᵀ\) |

本笔记固定：

```text
T = 3 个位置
d = 2 维
```

具体数字：

```text
位置 t=1:  q₁ = [1, 0]ᵀ ,  k₁ = [1, 0]ᵀ ,  v₁ = [2, 0]ᵀ
位置 t=2:  q₂ = [0, 1]ᵀ ,  k₂ = [0, 1]ᵀ ,  v₂ = [0, 3]ᵀ
位置 t=3:  q₃ = [1, 1]ᵀ ,  k₃ = [1, 1]ᵀ ,  v₃ = [1, 1]ᵀ
```

对应矩阵：

```text
Q = [[1, 0],
     [0, 1],
     [1, 1]]

K = [[1, 0],
     [0, 1],
     [1, 1]]

V = [[2, 0],
     [0, 3],
     [1, 1]]
```

输出里，位置 \(t\) 的结果记作向量 \(y_t\)（形状 \(d\times 1\)）。  
整段输出矩阵 \(Y\)（形状 \(T\times d\)）的第 \(t\) 行是 \(y_tᵀ\)。

---

## 2. 标准注意力里 \(Q Kᵀ V\) 在算什么

先算分数矩阵：

```text
A = Q Kᵀ          形状：T×T
```

例子：

```text
Kᵀ = [[1, 0, 1],
      [0, 1, 1]]

A = Q Kᵀ = [[1, 0, 1],
            [0, 1, 1],
            [1, 1, 2]]
```

含义：\(A_{t,s} = q_tᵀ k_s\)，即位置 \(t\) 对位置 \(s\) 的匹配分数。

再乘 value：

```text
Y = A V = (Q Kᵀ) V          形状：T×d
```

例子：

```text
Y = [[1, 0, 1],     [[2, 0],       [[3, 1],
     [0, 1, 1],  ×   [0, 3],   =    [1, 4],
     [1, 1, 2]]      [1, 1]]        [4, 5]]
```

所以：

```text
y₁ᵀ = [3, 1] ,  y₂ᵀ = [1, 4] ,  y₃ᵀ = [4, 5]
```

代价：\(Q Kᵀ\) 产生 \(T\times T\) 的矩阵。\(T\) 变大时，这块随 \(T²\) 变大。

（标准 Softmax 注意力还会在 \(A\) 上做逐行 Softmax；线性注意力把「相似度」换成可拆开的特征映射，核心代数仍是「先形成某种 \(Q\)–\(K\) 交互，再乘 \(V\)」。本节先盯住矩阵乘顺序本身。）

---

## 3. 线性注意力的重排：\(Q(Kᵀ V)\)

矩阵乘法满足结合律：

```text
(Q Kᵀ) V = Q (Kᵀ V)
```

左边：先做 \(Q Kᵀ\)（\(T\times T\)），再乘 \(V\)。  
右边：先做 \(Kᵀ V\)（\(d\times d\)），再左乘 \(Q\)。

继续用同一组数算右边。

### 3.1 先算状态矩阵 \(S = Kᵀ V\)

```text
Kᵀ 形状 d×T
V  形状 T×d
S = Kᵀ V  形状 d×d
```

例子：

```text
Kᵀ V = [[1, 0, 1],     [[2, 0],
        [0, 1, 1]]  ×   [0, 3],
                         [1, 1]]

      = [[1·2+0·0+1·1 , 1·0+0·3+1·1],
         [0·2+1·0+1·1 , 0·0+1·3+1·1]]

      = [[3, 1],
         [1, 4]]
```

### 3.2 再算 \(Y = Q S\)

```text
Y = Q (Kᵀ V) = Q S
```

例子：

```text
Y = [[1, 0],     [[3, 1],       [[3, 1],
     [0, 1],  ×   [1, 4]]   =    [1, 4],
     [1, 1]]                     [4, 5]]
```

与第 2 节的 \(Y\) **逐元素相同**。

### 3.3 两种顺序差在哪里

| 顺序 | 中间大矩阵 | 中间形状 |
|------|------------|----------|
| \((Q Kᵀ)V\) | \(Q Kᵀ\) | \(T\times T\) |
| \(Q(Kᵀ V)\) | \(Kᵀ V\) | \(d\times d\) |

序列很长、\(d\) 远小于 \(T\) 时，\(d\times d\) 的中间结果更省。  
这就是课上说的：线性注意力走 \(Q(Kᵀ V)\) 这条代数路径。

---

## 4. 把 \(Kᵀ V\) 拆成「一个位置加一项」

\(Kᵀ V\) 按列（按位置）展开。第 \(t\) 个位置贡献一项：

```text
外积：k_t v_tᵀ          形状：d×d
```

小例子核对 \(t=1\)：

```text
k₁ = [1, 0]ᵀ ,  v₁ᵀ = [2, 0]

k₁ v₁ᵀ = [[1],      [[2, 0]]  = [[2, 0],
          [0]]                   [0, 0]]
```

\(t=2\)：

```text
k₂ v₂ᵀ = [[0],      [[0, 3]]  = [[0, 0],
          [1]]                   [0, 3]]
```

\(t=3\)：

```text
k₃ v₃ᵀ = [[1],      [[1, 1]]  = [[1, 1],
          [1]]                   [1, 1]]
```

三项相加：

```text
[[2, 0],   [[0, 0],   [[1, 1],     [[3, 1],
 [0, 0]] +  [0, 3]] +  [1, 1]]  =   [1, 4]]
```

这正是前面的 \(S = Kᵀ V\)。

因此：

```text
Kᵀ V = Σ_{t=1}^{T}  k_t v_tᵀ
```

---

## 5. 递推形式：一个状态，从左往右更新

推理与因果训练里，位置 \(t\) 只聚合 \(s \le t\) 的 key/value。定义前缀状态：

```text
S_t = Σ_{s=1}^{t} k_s v_sᵀ
```

由求和得到递推：

```text
S_0 = 全零的 d×d 矩阵
S_t = S_{t-1} + k_t v_tᵀ
```

位置 \(t\) 的输出（行向量写法，与课上一致）：

```text
y_tᵀ = q_tᵀ S_t
```

（\(q_t\) 为列向量时，\(q_tᵀ S_t\) 是 \(1\times d\) 的行；它就是输出矩阵 \(Y\) 的第 \(t\) 行。）

### 5.1 逐步手算

**t = 1**

```text
S₁ = 0 + k₁ v₁ᵀ = [[2, 0],
                   [0, 0]]

y₁ᵀ = q₁ᵀ S₁ = [1, 0] [[2, 0],  = [2, 0]
                        [0, 0]]
```

**t = 2**

```text
S₂ = S₁ + k₂ v₂ᵀ = [[2, 0],   [[0, 0],   [[2, 0],
                    [0, 0]] +  [0, 3]] =  [0, 3]]

y₂ᵀ = q₂ᵀ S₂ = [0, 1] [[2, 0],  = [0, 3]
                        [0, 3]]
```

**t = 3**

```text
S₃ = S₂ + k₃ v₃ᵀ = [[2, 0],   [[1, 1],   [[3, 1],
                    [0, 3]] +  [1, 1]] =  [1, 4]]

y₃ᵀ = q₃ᵀ S₃ = [1, 1] [[3, 1],  = [4, 5]
                        [1, 4]]
```

完整前缀结果：

```text
y₁ᵀ = [2, 0]
y₂ᵀ = [0, 3]
y₃ᵀ = [4, 5]
S₃  = [[3, 1],   = 整段的 Kᵀ V
       [1, 4]]
```

### 5.2 与第 3 节并行公式的关系

第 3 节的 \(Y = Q(Kᵀ V)\) 让 **每个** \(q_t\) 都乘 **同一个** 最终 \(S_T = Kᵀ V\)，得到：

```text
[[3, 1],
 [1, 4],
 [4, 5]]
```

那是 **每个位置都看见整段** 的版本。  
第 5 节递推是 **每个位置只看见到自己为止的前缀** 的版本（因果）。  
训练若要与递推推理一致，并行实现里对「未来位置」加掩码，使位置 \(t\) 只用 \(S_t\)，两边的 \(y_t\) 相同。

## 6. 为什么说这「像 RNN」（你只需记住状态更新）

RNN 背景：**不必先学**。这里只用到一件事：

```text
每来一个新位置 t：
  1. 用旧状态 S_{t-1} 和新的 (k_t, v_t) 算出新状态 S_t
  2. 用 S_t 和 q_t 算出当前输出 y_t
  3. 把 S_t 留给下一步
```

状态 \(S_t\) 是固定形状 \(d\times d\) 的矩阵，与序列已经多长 **无关**。  
走完 \(T\) 步，仍只保存一个 \(d\times d\) 的 \(S_T\)。

逐步过程：

```text
S₀ = 0
t=1: S₁ ← S₀ + k₁v₁ᵀ ,  输出 ← q₁ 与 S₁
t=2: S₂ ← S₁ + k₂v₂ᵀ ,  输出 ← q₂ 与 S₂
t=3: S₃ ← S₂ + k₃v₃ᵀ ,  输出 ← q₃ 与 S₃
...
```

这就是课上的：

```text
S_t = S_{t-1} + k_t v_tᵀ
y_t = q_tᵀ S_t
```

## 7. 对偶：训练走并行，推理走串行

同一套数学（因果线性注意力）有两种算法外形：

### 7.1 并行外形（训练常用）

一次拿出整段 \(Q,K,V\)，按块做矩阵乘（实现上用掩码保证因果）：

```text
对每个 t：  y_tᵀ = q_tᵀ (Σ_{s≤t} k_s v_sᵀ)
```

GPU 上适合整段一起算。代数上仍围绕 \(d\times d\) 的聚合，以及按位置施加的因果掩码。

### 7.2 串行外形（推理常用）

按时间一步一步来：

```text
初始化 S = 0（d×d）
for t = 1..T:
    S ← S + k_t v_tᵀ
    y_tᵀ ← q_tᵀ S
```

生成第 \(t\) 个 token 时，只做一次 \(d\times d\) 的加法更新，再做一次 \(q_t\) 与 \(S\) 的乘。  
状态始终是一个 \(d\times d\) 的 \(S\)。

### 7.3 「对偶」四个字在说什么

```text
同一种 y_t 的定义
    ├─ 写成整段矩阵公式  → 便于训练时并行算
    └─ 写成 S_t 递推      → 便于推理时逐步算
```

两边算出的 \(y_t\) 相同（同一因果约定下）。  
课上那句「duality」指的就是：**一个公式，两种算法外形。**

## 8. 符号速查表

| 符号 | 是什么 |
|------|--------|
| \(T\) | 序列长度 |
| \(d\) | 头／通道维数（本笔记例子里是 2） |
| \(q_t,k_t,v_t\) | 位置 \(t\) 的 query / key / value（列向量） |
| \(Q,K,V\) | 由全体 \(q_tᵀ,k_tᵀ,v_tᵀ\) 堆成的 \(T\times d\) 矩阵 |
| \(Q Kᵀ\) | \(T\times T\) 分数矩阵；条目 \(q_tᵀ k_s\) |
| \(Kᵀ V\) | \(d\times d\) 聚合矩阵；等于 \(\Sigma_t k_t v_tᵀ\) |
| \(S_t\) | 前缀聚合 \(\Sigma_{s\le t} k_s v_sᵀ\)；递推里的「状态」 |
| \(y_t\) | 位置 \(t\) 的输出向量；因果下 \(y_tᵀ = q_tᵀ S_t\) |

---

## 9. 一条线串起来

```text
1. 注意力输出由 Q、K、V 经矩阵乘得到。
2. (Q Kᵀ)V 与 Q(Kᵀ V) 在数字上可以相同（结合律）。
3. Kᵀ V 是 d×d，并且等于每个位置外积 k_t v_tᵀ 的和。
4. 把这个和写成前缀：S_t = S_{t-1} + k_t v_tᵀ。
5. 因果输出：y_tᵀ = q_tᵀ S_t。
6. 同一定义：训练可用整段并行实现；推理可用 S_t 一步一步更新。
```

课上那三行，对应的就是第 4–5 步；「像 RNN」指的是第 4 步那种 **固定大小状态、逐步更新** 的外形。
